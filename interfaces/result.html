Interface Result<T, E>

Manages access to a value that represents either success (Ok) or failure (Err).

let result: Result<string, number>;

// A value for success.
result = ok('success');

// A value for failure.
result = err(123);

Type parameters

  • T

    The type of the success value.

  • E

    The type of the failure value.
